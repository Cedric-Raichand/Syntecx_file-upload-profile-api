# 🖼️ File Upload API with Metadata

A RESTful API built using Node.js, Express, MongoDB, and Multer that allows users to upload, update, delete, and retrieve profile pictures with stored file metadata.

---

## 🚀 Features

- Upload profile pictures
- Store file metadata in MongoDB
- Associate images with user accounts
- Update profile pictures
- Delete profile pictures
- Serve uploaded images via URL
- Restrict file types
- File size validation

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- Multer
- dotenv
